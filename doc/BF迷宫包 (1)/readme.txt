■项目简介■

BF迷宫包是一个轻量级的AS3迷宫处理类库。

此类库提供了迅速生成迷宫和迷宫求解的方法，以及便捷的迷宫修饰和分析功能，让您可以按需要轻松地创造出丰富多样的迷宫。
我创建这个类库是为了在自己的项目中使用，但您同样可以免费使用它，因为它是开源共享的——只要您保留白蓝紫的署名即可。

主要功能：
·风格多样的迷宫生成器，应对不同需求
·高效低耗的通用A*寻路，用户体验更佳
·功能丰富的迷宫滤镜类，让地形更自然
·便捷实用的数据分析器，岛屿一目了然
·全面地支持随机数种子，方便重现复用

■更新履历■

2012-11-15 Ver1.00
---------------------------
*正式版，也就意味着不会再频繁更新了
*增加了迷宫分析（MazeAnalyzer）类，提供了生成岛数组等数据分析功能
*迷宫滤镜类添加了加权替换（replaceWeightedFlag）函数，便于根据连通情况赋值
*迷宫滤镜类添加了批量替换（replaceMultipleFlag）函数，便于根据内容分配权重
*优化了迷宫生成类和迷宫滤镜类，性能略有提升

2012-11-04 Ver0.03
---------------------------
*增加了迷宫求解（MazeSolver）类，提供了高效的A*寻路功能
*所有的源码都划归到了bf.maze包下，这是为了配合日后的规划

2012-10-31 Ver0.02
---------------------------
*增加了迷宫滤镜（MazeFilter）类，以便对迷宫数组进行各种便捷的修饰
*深度优先改名为递归回溯（RBMazeCreater），这主要是出于严谨性的考虑
*修复了随机普里姆类无法生成无边框迷宫的bug
*优化了迷宫生成类，性能明显提升：递归分割0.7%up，递归回溯8.5%up，随机普里姆34%up
*更新了demo的界面，便于查看性能

2012-10-25 Ver0.01
---------------------------
*第一个版本
*提供了递归分割（RDMazeCreater）、深度优先（DFSMazeCreater）、随机普里姆（RPMazeCreater）三种风格的迷宫生成类
